# Roadmap

## Alpha hardening

- Automated CPU, NVIDIA, AMD, and Intel hardware test matrix.
- Streaming token previews and more accurate first-token benchmarks.
- Crash recovery for runs interrupted by an application restart.
- Broader real-machine coverage for Ollama and LM Studio model installation.
- Router rule editor and richer branch visualization.
- Signed installer, verified updater, and release automation.

## Later

- Developer Mode sandbox research for custom code.
- Additional local embedding and document-index options.
- Additional opt-in cloud provider adapters behind provider-neutral interfaces.
- Windows ARM64 support.

Public internet webhooks, broad approval bypasses, and automatic exposure of
local inference servers to the LAN are intentionally not planned for v1. The
narrow automatic-email node remains an explicit workflow-author choice.
